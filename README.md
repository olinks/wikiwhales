# Project Title

A Dapp for Getting all Token Holders of a Particular token

## Installation

Front End

```bash
  cd wikiwhales
  cd frontend
  npm install
  npm run start
```

Back End

```bash
  cd wikiwhales
  cd backend
  npm install && npm run start
```

## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express
